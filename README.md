## 📌 Sobre

Este repositório tem como objetivo compartilhar aprendizados, desafios e projetos criados com a linguagem Go. Aqui você encontrará desde scripts simples até projetos mais robustos, sempre com foco em **boas práticas**, **simplicidade** e **performance**, pilares da linguagem.

---

## 🛠️ Tecnologias e Ferramentas

- [Go](https://golang.org/) — Linguagem principal
- [Gin](https://github.com/gin-gonic/gin) — Framework para APIs REST
- [GORM](https://gorm.io/) — ORM para bancos de dados
- [Docker](https://www.docker.com/) — Para ambientes isolados
- [PostgreSQL](https://www.postgresql.org/) — Banco de dados relacional
- [Visual Studio Code](https://code.visualstudio.com/) — Editor de código
